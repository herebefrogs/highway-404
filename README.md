Highway 404
====================

My game entry for JS13K Games 2020 on the theme "404".

Getting Started
---------------

```
npm start
-> build game, open a browser, watch source and livereload browser on changes

npm run build
-> build game for gamejam submission (no sourcemap and livereload script)
```
